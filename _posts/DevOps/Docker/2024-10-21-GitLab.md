***gitlab 컨테이너 접속 후 파일 수정***
```bash
# gitlab 컨테이너 접속
sudo docker exec -it gitlab /bin/bash

vi /etc/gitlab/gitlab.rb 

gitlab-ctl reconfigure
```
> external_url 설정 시, 포트 빼고 적을 것  
> 포트는 docker-compose.yml 파일내 설정으로 충분
